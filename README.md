# 🚀 Itzfizz Web Assignment

A modern multi-page animated website built using **HTML, CSS, and JavaScript**, featuring smooth animations powered by **GSAP** and **ScrollTrigger**.

---

## ✨ Features

* 🎬 Smooth GSAP animations
* 📜 Scroll-based effects (ScrollTrigger)
* 📄 Multi-page navigation (Home, About, Projects)
* 🎨 Clean dark UI design
* ⚡ Lightweight and fast

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP (GreenSock Animation Platform)
* ScrollTrigger

---

## 📂 Project Structure

```
itzfizz-assignment/
│
├── index.html
├── about.html
├── projects.html
├── styles.css
├── script.js
```

---

## ⚙️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Or use VS Code Live Server
